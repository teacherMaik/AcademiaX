HTML5

Al comenzar este Bootcamp, ya tenía bastantes conocimientos de esta herramienta, habiendo programado juegos y webs. Aún así siempre hay algo que aprender y acá comparto mis ejercicios.

Retos del curso

  • Texto
  • Navigación (este mismo menú)
  • Imágenes
  • Multimedia
  • Contenido (esta misma página)
  • Estructura
  • Formularios

De este curso destacaría conocer el uso de citas, etiquetas para código, srcsets para imágenes, estructuración de datos con tablas y los diferentes inputs y sus atributos para los formularios.

Me gustaría aprender más sobre accesibilidad, en específico los atributos aria-label, y diseñar mejores experiencias de usuario.
